#  LLM-Based ATS & Resume Builder

An intelligent Applicant Tracking System (ATS) and Resume Builder powered by local LLMs (Ollama). No API keys, no subscriptions - completely free and private!

##  Features

- **Resume Analysis**: Get detailed feedback on your resume
- **ATS Scoring**: Compare your resume against job descriptions
- **Smart Suggestions**: AI-powered improvement recommendations
- **Resume Generation**: Auto-create tailored resumes from your base resume
- **Application Tracking**: CSV-based tracking of all applications
- **Multiple Formats**: Support for PDF, Word, and text resumes
- **Background Context**: Provide your background for better recommendations
- **Complete Privacy**: Runs locally with no data leaving your machine
